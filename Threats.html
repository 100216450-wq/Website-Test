Welcome to the threats page

Malware:

Malware is caused by clicking on phishing emails that carry different types of dangerous software that causes massive problems for the user. Malware can spread through shared files or using the same network, this causes other users devices to get infected with malware. If the computer isn't protected by a firewall then the computer is at risk of losing sensitive data.

Ransomware

Ransomware is a dangerous threat that could cause the users data to be stolen and then the user has to pay the ransom to get back that data for a large amount of money. Ransomware could be prevented by doing regular backups on the computer this makes copies of the sensitive information so that the user doesn't have to pay the ransom. Backups are stored an external drive to keep the data safe if the users system gets attacked.

Viruses

Virus are the most dangerous threat that could happen to a computer if it becomes infected because when the virus is already in the computer it can't be fixed and when the infected computer sends a file or sharing a network it will spread onto other devices and cause big problems with the computer.

Spyware

Spyware is rarely seen but it is a dangerous threat because spyware collects passwords and information that is sent to the attacker so they can use it for personal gain and gain unauthorized data.

USB sticks

USB sticks can carry the most dangerous software that will install vriuses and other malware onto the computer and give the attacker full access to the computer. To prevent this use a firewall that blocks all of the ports to make sure that no dangerous threats try to plug in the computer pors.
